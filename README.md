A simple and elegant web application to track penalty points for pool games (or any game where the lowest score wins).

## Features

-   **Player Setup**: Support for 2 to 4 players with custom names.
-   **Score Tracking**: Easily add penalty points to each player's score.
-   **Undo Functionality**: Made a mistake? Undo the last added score with a single click.
-   **Responsive Design**: Works great on mobile devices and desktops.
-   **Game Management**: Reset scores or start a completely new game with ease.
-   **Visual Feedback**: Animations for score updates and interactions.

## How to Use

1.  **Setup**: Enter the names of the players (minimum 2).
2.  **Start Game**: Click "Start Game" to enter the tracking view.
3.  **Add Points**: Enter the penalty points in the input field for a player and click the `+` button.
4.  **Undo**: Click the `‚Ü©Ô∏è` button next to the input field to remove the last added points.
5.  **Reset/New Game**: Use the buttons at the bottom to reset the current scores or start over with new players.
